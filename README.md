PHP学习路线图

* **PHP基础知识** - 语法、变量及类型、数组、循环控制、函数

* **HTML知识** - HTML标签、CSS演示、Javascrip脚本

* **WEB服务器搭建** - Linux基础、LNMP安装、PHP-FPM配置

* **PHP MySQL操作** - CRUD操作、表单处理、PDO

* **PHP面向对象** - 类、对象、继承、封装、多态、性状，以及免面向对象设计原则

* **设计模式** - 类之间的关系和UML类图、主要的设计模式

* **代码标准** - PHP-FIG PSR1~PSR4代码规范

* **版本控制** - git版本控制和github代码仓库

* **PHP高级功能** - Cookie、Session、JSON、XML、AJAX、文件读写、上传、图片处理、RESTful

* **MVC框架** - 单一入口、MVC三部分的职责划分、PHP框架

* **组件** - composer、组件使用

* **Nginx** - Nginx配置优化、集群、CDN分布

* **数据库** - ORM、索引、全文索引、锁机制、分表、集群、Redis缓存

* **PHP优化** - OPcache、XDebug、PHPUnit单元测试

另外就是，多学习别人的代码，多自己写代码，多总结。
